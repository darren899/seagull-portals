How to use this signature

Gmail (recommended):
1. Fill in your details above — the preview updates live.
2. Click Copy for Gmail. The signature image is now on your clipboard.
3. In Gmail, click the gear → See all settings → General → scroll to Signature.
4. Click inside the signature editor and press Ctrl+V (or Cmd+V on Mac).
5. Scroll to the bottom and click Save changes.

Outlook:
1. Click Download PNG for Outlook for a high-resolution image.
2. File → Options → Mail → Signatures → Insert Picture → select your PNG.
3. Resize to roughly 600px wide if needed.

Copy HTML (advanced): Copies an editable text-based signature (no logo graphic, just text and styling). Use only if you specifically need an HTML signature — the Copy for Gmail button gives a better result for most people.
